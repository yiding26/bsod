# bsod

A program can make Windows B(lue)S(creen) O(f) d(earth).

一个可以使 Windows 蓝屏的程序。

```
bsod [code] [time] [showOrHide]

[code]       stop code
             蓝屏代码
             (c0000000-cfffffff)

[time]       Wait time (default is 0)
             等待时间 （默认是0）

[showOrHide] 1 is to show the console, 0 is to hide the console (default is 0)  
             1是显示控制台，0是隐藏控制台（默认是0）
```
